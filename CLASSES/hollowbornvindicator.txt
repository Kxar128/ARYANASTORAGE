How to get:

Start at Ultragramielhub

Two options

1. Easy version
- "Echoes of a Forgotten Fate (Weekly)" 
  - 4 x Death's Power -> /join shadowattack --> Kill Death (Monster)
  - 300 x Hollow Soul --> 
    - /join hbchallenge --> Kill The Darkdweller (With Party)
    - /join Dawnsanctum --> Accept "Nauthiz - Realized" quest from "Veronika (NPC) (If solo)
  - 40 x Vindicator badge --> /join Trygve --> Accept "Nauthiz" quest from "Lae (NPC)"
  - 100 x Grace Orb --> /join neofortress --> Accept "Grace Extraction" quest from "Lae (NPC)"
  - 60 x Gramiel's Emblem --> /join dawnsanctum --> Kill Celestial Gramiel in room 10
  - 20 x Vindicator's Crest --> /join neotower -> Accept "Mutual Friends" quest from "Lae (NPC)"
  - 20 x Gramiel the Graceful's Insignia --> /join ultragramiel --> Accept "Resuscitated Hope" quest from "Lae (NPC)" and kill boss at the same map

2. Hard version

- "Trials of the Vessel (Weekly)" 
  - 4 x Death's Power --> /join shadowattack --> Kill Death (Monster)
  - 6000 x Hollow Soul ->
    - /join hbchallenge --> Kill The Darkdweller (With Party)
    - /join Dawnsanctum --> Accept "Nauthiz - Realized" quest from "Veronika (NPC) (If solo)
  - 800 x Vindicator badge --> /join Trygve --> Accept "Nauthiz" quest from "Lae (NPC)"
  - 1600 x Grace Orb --> /join neofortress --> Accept "Grace Extraction" quest from "Lae (NPC)"
  - 1200 x Gramiel's Emblem --> /join dawnsanctum --> Kill Celestial Gramiel in room 10
  - 400 x Vindicator's Crest --> /join neotower -> Accept "Mutual Friends" quest from "Lae (NPC)"
