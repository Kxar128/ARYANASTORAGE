🔥 CLASS COMBO — HOW TO USE
╭────────────────────────────────────────────────────────╮

  Rotation:  2 → 2 → 2 → 3 → 2 → 2 → 1 → 3 → 2 → 5

  ──────────────────────────────────

  Step-by-step:

    1. Use Skill 2 three times to build stacks / apply pressure.
       [ 2 → 2 → 2 ]

    2. Fire Skill 3 to trigger the mid-combo burst or debuff window.
       [ 3 ]

    3. Continue with Skill 2 twice more to sustain pressure.
       [ 2 → 2 ]

    4. Use Skill 1 — the pivot / reset point of the rotation.
       [ 1 ]

    5. Follow immediately with Skill 3 to maximize the window
       opened by Skill 1.
       [ 3 ]

    6. One more Skill 2 to bridge into the finisher.
       [ 2 ]

    7. Close with Skill 5 — the finisher. Use at peak stacks
       for maximum damage.
       [ 5 ]

  ──────────────────────────────────

  ⚠️  NOTE: Do not skip the 1 before the second 3 — it sets up
      the finisher window. Repeat rotation once Skill 5 is
      off cooldown.
